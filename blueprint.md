# AI GIF Tournament Blueprint

## 1. 개요 (Overview)

본 프로젝트는 사용자가 AI가 생성한 GIF 이미지들을 1:1 토너먼트 방식으로 비교하며 가장 마음에 드는 이미지를 선택하는 웹 애플리케이션입니다. 최종 우승 이미지가 결정되면, 사용자는 선택 이유에 대한 피드백을 남길 수 있습니다. 수집된 피드백은 향후 더 나은 AI 이미지 생성을 위한 데이터로 활용됩니다. 애플리케이션은 한국어, 영어, 일본어, 중국어를 지원하여 다국어 사용자를 대상으로 합니다.

## 2. 프로젝트 설계 (Project Outline)

### 핵심 기능 (Core Features)

-   **토너먼트 진행**: 8개의 GIF 이미지로 8강 토너먼트를 진행합니다. (추후 확장 가능)
-   **랜덤 대진**: 매 라운드마다 남은 이미지 중에서 랜덤으로 두 개를 선택하여 보여줍니다.
-   **승자 기록**: 각 라운드에서 선택된 이미지는 다음 라운드로 진출합니다.
-   **최종 우승자 표시**: 모든 라운드가 끝나면 최종 우승 이미지를 화면에 표시합니다.
-   **피드백 제출**: 우승자 화면에서 사용자는 피드백을 작성하고 제출할 수 있습니다.
-   **피드백 저장**: 제출된 피드백은 `feedback.log` 파일에 누적하여 저장됩니다.

### 사용자 인터페이스 (UI)

-   `index.html`을 기본 구조로 사용합니다.
-   **헤더**: 언어 선택 버튼 (한국어, 영어, 일본어, 중국어)을 포함합니다.
-   **메인 영역**:
    -   토너먼트 진행 중: 두 개의 GIF 이미지가 나란히 표시되며, 각 이미지 하단에는 생성 AI의 이름(예: DALL-E, Nano Banana)이 표시됩니다.
    -   토너먼트 종료 후: 최종 우승 이미지와 피드백 입력 폼이 표시됩니다.
-   **푸터**: 간단한 저작권 정보를 표시합니다.

### 스타일 및 디자인 (Style & Design)

-   `style.css`에서 모든 스타일을 관리합니다.
-   현대적이고 직관적인 디자인을 적용합니다.
-   부드러운 화면 전환을 위해 CSS 애니메이션을 활용합니다.
-   모바일 반응형 디자인을 적용하여 모든 기기에서 최적의 경험을 제공합니다.
-   CSS 변수 (Variables)를 사용하여 색상, 폰트 등 디자인 요소를 체계적으로 관리합니다.

### 데이터 구조 (Data Structure)

-   `main.js`에서 이미지 데이터를 관리합니다.
-   이미지 정보는 아래와 같은 객체 배열로 구성합니다.

    ```javascript
    const images = [
      { id: 1, src: 'images/1.gif', ai: 'Nano Banana' },
      { id: 2, src: 'images/2.gif', ai: 'DALL-E' },
      // ... more images
    ];
    ```

### 다국어 지원 (Internationalization - i18n)

-   `main.js` 내에 언어별 텍스트를 객체 형태로 저장합니다.
-   사용자가 언어를 선택하면 해당 언어의 텍스트로 UI를 동적으로 변경하는 함수를 구현합니다.

    ```javascript
    const i18n = {
      ko: { round: '8강', winner: '최종 우승!', ... },
      en: { round: 'Quarter-Finals', winner: 'Final Winner!', ... },
      // ... other languages
    };
    ```

### 피드백 (Feedback)

-   사용자가 피드백을 제출하면 `fetch` API를 사용하여 서버(백엔드 기능 구현 시) 또는 파일 시스템(초기 버전)에 로그를 남깁니다.
-   초기 버전에서는 `default_api.write_file`을 사용하여 `feedback.log` 파일에 피드백을 추가하는 방식으로 구현합니다.

## 3. 현재 작업 계획 (Current Plan)

1.  **`blueprint.md` 파일 생성**: 프로젝트의 청사진을 정의합니다. (완료)
2.  **프로젝트 구조 생성**: `images` 폴더를 생성하고, 기본 `index.html`, `style.css`, `main.js` 파일을 생성합니다.
3.  **HTML 기본 구조 작성**: 토너먼트 UI와 최종 결과 UI의 기본 마크업을 `index.html`에 작성합니다.
4.  **CSS 기본 스타일링**: `style.css`에 레이아웃, 색상, 폰트 등 기본 디자인을 적용합니다.
5.  **JavaScript 초기 로직 구현**: `main.js`에 이미지 데이터 로드, 토너먼트 시작 및 라운드 진행 로직을 구현합니다.
6.  **다국어 기능 구현**: 언어 변경 기능을 구현하고 UI 텍스트를 동적으로 변경합니다.
7.  **피드백 기능 구현**: 피드백 폼을 만들고 제출 시 파일에 저장하는 기능을 구현합니다.
